# rust-todo

A RESTful api for todo list app implement in Rust

----------------------

## Usage
Install `Rust` and `git` and run this:
```bash
$ git clone https://github.com/IamRezaMousavi/rust-todo.git
$ cargo run
```

### for change port
```bash
$ cargo run -- -p <port_number>
```

### for build
```bash
$ cargo build --release
```
